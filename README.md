# Ambient Manager

A focused overview of the core capabilities currently supported by Ambient Manager.

- **List all environments in the sidebar** so teams can quickly scan everything that is available.
- **Favorite environments** to pin the ones you open most often.
- **Filter tasks by environment** to narrow work items to a specific context.

## Installation (Chrome, unpacked)

1) Open `chrome://extensions`.
2) Enable **Developer mode** in the top-right corner.
3) Click **Load unpacked** and select the project directory for this extension.
4) Confirm the extension appears in the list and is enabled.
